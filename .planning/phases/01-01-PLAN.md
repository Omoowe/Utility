### Task 3: Calculator Logic (Days 3-4, 8 hours)

**Wave:** 2 | **Depends On:** [Task 2] | **Blocks:** [Task 5]

**Objective:** Implement calculation functions for all 7 calculators using decimal.js for precision.

**Calculators:**

1. **Mortgage Calculator**
   - Formula: M = P[r(1+r)^n]/[(1+r)^n-1]
   - Inputs: Loan amount, annual interest rate, term (years)
   - Outputs: Monthly payment, total paid, total interest
   - Precision: decimal.js (prevents 0.1 + 0.2 ≠ 0.3 errors)

2. **Paycheck Calculator**
   - Formula: Gross pay - federal tax (bracket lookup) - FICA (7.65%) - state tax (~5%)
   - Inputs: Annual salary, pay frequency (annual/monthly/biweekly/weekly)
   - Outputs: Gross pay, federal tax, FICA, state tax, net pay (all per pay frequency)
   - Note: Use simplified federal brackets + average state tax (Phase 2: state-specific variations)

3. **ROI Calculator**
   - Formula: ROI = (Gain - Cost) / Cost × 100
   - Inputs: Initial investment, gain/loss, period (years)
   - Outputs: ROI %, annualized ROI, final value

4. **Loan Calculator**
   - Formula: Same as mortgage, but handles extra principal payments
   - Inputs: Loan amount, annual rate, term, optional extra payment
   - Outputs: Monthly payment, total paid, total interest, payoff months

5. **Investment Calculator**
   - Formula: A = P(1+r/n)^(nt) + PMT × [((1+r/n)^(nt) - 1) / (r/n)]
   - Inputs: Principal, monthly contribution, annual rate, years
   - Outputs: Final amount, total contributions, total interest earned

6. **Age Calculator**
   - Formula: Today - birth date = years, months, days
   - Inputs: Birth date
   - Outputs: Age (years, months, days), days to next birthday

7. **Random Number Generator**
   - Formula: Math.floor(Math.random() * (max - min + 1)) + min
   - Inputs: Min, max, count
   - Outputs: Array of random integers, copy to clipboard

**Central Registry: tools.ts**
- Interface: Tool { slug, name, category, description, keywords, title, metaDescription, h1, intro, howItWorks, interpretationGuide, faqs, relatedTools, calculator }
- Array: 7 tool definitions with metadata + calculator config
- Functions: getTools(), getTool(slug)

**SEO Utilities: seo.ts**
- generateCalculatorSchema(tool) - Returns Calculator JSON-LD
- generateFAQSchema(tool) - Returns FAQPage JSON-LD with all Q&As

**Verification:**
- npm run type-check passes

  - Paycheck: $50k annual → ~$3,200/month gross, ~$2,500/month net ✓
  - ROI: $10k invested, $2k gain, 2 years → 20% ROI ✓

**Done:** All 7 calculators logic complete. decimal.js for precision. Central metadata registry. Ready for content.
